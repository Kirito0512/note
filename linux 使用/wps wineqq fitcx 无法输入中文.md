# fitcx 无法输入中文

## 解决方法
可以直接修改desktop配置文件

```
sudo vim /usr/share/applications/wps-office-et.desktop
```

把其中的

```
Exec=/usr/bin/et %f
```

修改为:

```
Exec=env XIM="fcitx" XIM_PROGRAM="fcitx" XMODIFIERS="@im=fcitx" GTK_IM_MODULE="fcitx" QT_IM_MODULE="fcitx" /usr/bin/et %f
```
