# onvif

## rtsp 授权
通过 onvif 获取到 rtsp 链接，但是访问不到视频，通过 onvif 获取到的截图 url 获取截图时，需要 digest 授权。

rtsp 通过用户名密码获取授权的方式如下：
```
rtsp://username:password@[camera-address]
```
