# apidoc 安装
## 安装 nodejs
[nodeJs 下载地址](http://nodejs.cn/download/)

## npm 配置淘宝镜像
```
npm config set registry https://registry.npm.taobao.org
```
## 安装 apidoc
```
npm install apidoc -g
```

## 使用 apidoc
```
apidoc -i E:\code -o E:\api
```
apidoc 详细使用说明，见[参考链接](http://apidocjs.com/)。
